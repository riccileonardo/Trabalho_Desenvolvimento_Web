Projeto de Desenvolvimento Web ChaveMestra

O projeto ChaveMestra visa criar uma maneira de armazenamento de senhas de maneira simples e fácil. Ao se cadastrar e ao usar nosso produto, é possível gerenciar suas senhas de maneira totalmente segura.

O projeto foi desenvolvido por alunos da Universidade do Estado de Mato Grosso (UNEMAT) e tem como objetivo facilitar a vida dos usuários que desejam armazenar suas senhas de maneira segura e confiável. Feito para FACET-SNP-307 - DESENVOLVIMENTO WEB - T01 (2024.1 - 4N123) como trabalho referente a terceira nota.
